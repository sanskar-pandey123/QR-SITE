Advocate High Court Website

1. Project Overview
This is a professional website developed for a High Court Advocate to showcase legal services, experience, and practice areas. The website helps clients understand the advocate’s expertise and contact them easily.

2. Features
Advocate profile with qualifications and experience
Practice areas (Civil, Criminal, Family, Consumer, Labour, Revenue, Banking Law)
Detailed case and service information
Responsive contact form
Testimonial section
Mobile-friendly and responsive design
Clean and professional user interface

3. Technologies Used
HTML
CSS
JavaScript
PHP
MySQL

4. Purpose of the Project
To create a professional online presence for a High Court Advocate
To provide clear legal service information to clients
To improve accessibility and communication
5. How to Run the Project
Install XAMPP
Place project folder inside htdocs
Start Apache and MySQL
Open browser and type localhost/project-folder-name
